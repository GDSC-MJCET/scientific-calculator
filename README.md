# Scientific Calculator

## Objective

Build a scientific calculator web application capable of performing both basic and advanced mathematical operations.

## Requirements

The calculator should support:

### Basic Operations

- Addition
- Subtraction
- Multiplication
- Division

### Scientific Functions

- Square root
- Power/exponents
- Trigonometric functions
- Logarithms
- Percentage

## Technical Requirements

- Use HTML, CSS, and JavaScript
- Create a clean and responsive UI
- Handle invalid inputs properly
- Organize files properly

## Suggested Features

- Keyboard support
- Dark/light mode
- Calculation history
- Memory functions

## Submission Requirements

- Upload the complete project folder
- Ensure all functions work correctly
- Push the project to GitHub
- Include proper code formatting

## Evaluation Criteria

- Accuracy of calculations
- User interface
- Responsiveness
- Code structure
- Additional features implemented
